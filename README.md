# 📚 EduMindAI

EduMindAI is an Offline Hinglish Study Assistant built using Llama 3.2, Retrieval-Augmented Generation (RAG), FAISS, and Streamlit.

## Features

- 📄 PDF Question Answering
- 🤖 Llama 3.2 (1B)
- 🔍 Semantic Search using FAISS
- 🇮🇳 Hinglish Responses
- 💻 Fully Offline

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3.2 1B
- Sentence Transformers
- FAISS
- PyMuPDF

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
